# Expected Value (EV) Improvement Backlog

실행 후보는 실제 자본 리스크 없이 검증 가능한 `shadow-only` 모드 우선으로 정렬되었습니다.

## 1. [Shadow-Only] Fallback 진입 모멘텀 필터 도입 (손실 패턴 1번 방어)
- **예상 기대값 개선 축:** Fallback 진입 후 발생하는 Soft Stop 손실(-41.18%)의 절반 이상 방어. 전체 승률 약 5~8% 상승 예상.
- **리스크:** Fallback 진입 후 늦게 터지는 슈팅(수익 패턴 1번) 기회까지 함께 차단될 위험(False Positive).
- **검증 지표:** Shadow 환경에서 필터 적용 시 차단된 거래 중 실제 상승한 비율(Missed Upside) vs 하락 방어 비율 측정.

## 2. [Shadow-Only] Partial Fill 직후 시간제한 탈출 로직 (손실 패턴 2번 방어)
- **예상 기대값 개선 축:** 부분 체결 후 돌파 실패로 인한 -1.6% 대의 손실을 -0.3% 수준의 본절/약손절로 방어.
- **리스크:** 일시적 눌림(Whipsaw)에 너무 일찍 털려버리고 이후 날아가는 파동을 놓칠 위험.
- **검증 지표:** Partial Fill 후 '3초 내 추가 체결 또는 상승' 조건 만족 여부에 따른 1분/3분 후 MFE(Maximum Favorable Excursion) 비교.

## 3. [Shadow-Only] 강한 돌파 시 Partial Fill 후 Split-Entry(불타기) 추가 (수익 패턴 2번 극대화)
- **예상 기대값 개선 축:** 부분 체결로 인해 적게 먹고 있는 수익 거래의 비중을 확대하여 총 기여 수익 30% 이상 증대.
- **리스크:** 추격 매수가 되어 평단가가 높아지고, 고점 꼬리에서 물리며 손실로 전환될 위험.
- **검증 지표:** Split-Entry 가상 주문 생성 후, 합산 평단가 대비 실제 청산가 기준의 시뮬레이션 수익률.

## 4. [Canary] AI Momentum Decay 민감도 상향 (손실 -> 수익 패턴 전환)
- **예상 기대값 개선 축:** 기존 Soft Stop까지 기다렸다가 -1.6% 손절하는 케이스들을 모멘텀 감소 선제 인식으로 +0.5% 익절이나 -0.5% 약손절로 전환.
- **리스크:** 과민 반응하여 정상적인 추세 중의 사소한 호가 공백에도 익절해버려 평균 수익률(Avg Profit Rate)을 갉아먹을 수 있음.
- **검증 지표:** Canary 봇에서 모델 민감도를 15% 올린 병렬 모델 구동 후, Exit 시그널 발생 시점과 실제 최적 매도 시점(Sell Bucket) 간의 오차 비교.

## 5. [Shadow-Only] AI Threshold Dynamic 완화 (기회비용 회수)
- **예상 기대값 개선 축:** 하루 20만 건씩 차단되는 시그널 중 초강세 거래량 동반 종목에 한해 예외 진입을 허용하여 일일 거래 기회(Submitted) 2배 이상 확보.
- **리스크:** 뇌동 매매 증가 및 휩소 장세에서 대규모 연쇄 손실 발생.
- **검증 지표:** Shadow 환경에서 AI Score 기준을 5점 낮추되, '10틱 체결강도 150% 이상' 조건을 AND로 묶었을 때의 가상 거래 승률 및 손익비(Profit Factor) 확인.
